# Single-Page-HTML-CV

Project Live URL: https://RyanSkg.github.io/Curriculo-em-HTML/blob/main/Single-Personal-CV

Este projeto é um currículo vitae (CV) de uma página, construído com HTML5 semântico. Ele demonstra o uso correto de elementos estruturais de HTML para criar um layout de currículo claro, acessível e bem organizado.

O CV inclui seções essenciais como perfil, formação acadêmica, habilidades e histórico profissional, e é estruturado para ser facilmente estilizado em projetos futuros usando CSS.

Para melhorar a visibilidade e o compartilhamento, o projeto implementa meta tags de SEO, tags Open Graph (OG) para pré-visualizações em mídias sociais e um favicon para melhor identificação pelo navegador.

✅ Key Features
```
Semantic HTML5 structure (header, main, section, article, footer)

Single-page CV layout

SEO-friendly meta tags

Open Graph tags for social media sharing

Favicon support

Clean structure ready for future styling
```

HTML5
